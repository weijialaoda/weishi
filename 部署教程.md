# 📱 记账 APP 部署教程

## 快速部署（推荐 Vercel）

### 方案一：Vercel 部署（最简单，5 分钟搞定）

#### 步骤 1：准备文件
你已经有了完整的部署文件包，包含：
- `index.html` - 主页面
- `mobile.html` - 手机版优化页面

#### 步骤 2：访问 Vercel
打开浏览器访问：**https://vercel.com**

#### 步骤 3：注册/登录
- 可以用 GitHub、GitLab 或邮箱注册
- 推荐使用 GitHub 账号，更方便管理

#### 步骤 4：创建项目
1. 点击 **"Add New Project"**
2. 选择 **"Deploy Third Party Framework"** 或直接拖拽文件
3. **更简单的方式**：安装 Vercel CLI

#### 步骤 5：使用 CLI 部署（最快）

**Windows 用户：**
```bash
# 安装 Node.js（如果还没有）
# 访问 https://nodejs.org 下载安装

# 安装 Vercel CLI
npm install -g vercel

# 进入项目目录
cd expense_tracker

# 部署
vercel
```

第一次使用会提示登录，按提示操作即可。

#### 步骤 6：获取访问地址
部署成功后，Vercel 会给你一个网址，格式类似：
```
https://your-project-name.vercel.app
```

**在手机上使用：**
1. 复制这个网址
2. 在手机浏览器打开
3. 点击浏览器菜单 → "添加到主屏幕"
4. 就可以像 APP 一样使用了！

---

### 方案二：Netlify 部署（同样简单）

#### 步骤 1：访问 Netlify
打开：**https://netlify.com**

#### 步骤 2：拖拽部署
1. 注册/登录 Netlify
2. 进入 Dashboard
3. 找到 **"Sites"** 标签
4. **直接把 `expense_tracker` 文件夹拖到网页上**
5. 等待上传完成

#### 步骤 3：获取网址
部署完成后，Netlify 会给你一个网址：
```
https://your-site-name.netlify.app
```

你可以自定义这个名称。

---

### 方案三：GitHub Pages（适合有 GitHub 账号的用户）

#### 步骤 1：创建仓库
1. 访问 **https://github.com**
2. 登录账号
3. 点击 **"New repository"**
4. 命名仓库，如 `expense-tracker`
5. 设为公开（Public）
6. 点击 **"Create repository"**

#### 步骤 2：上传文件
**方式 A：网页上传**
1. 点击 **"uploading an existing file"**
2. 把 `index.html` 和 `mobile.html` 拖进去
3. 点击 **"Commit changes"**

**方式 B：使用 Git 命令**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/expense-tracker.git
git push -u origin main
```

#### 步骤 3：启用 GitHub Pages
1. 进入仓库的 **Settings**
2. 找到 **"Pages"** 菜单
3. Source 选择 **"main branch"**
4. 点击 **Save**

#### 步骤 4：获取网址
几分钟后，你的网站就会上线：
```
https://你的用户名.github.io/expense-tracker/
```

## 📱 手机端使用方法

### 1. 访问网站
在手机浏览器输入部署后获得的网址

### 2. 添加到桌面

**iPhone (Safari)：**
1. 点击底部工具栏的 **分享按钮**（方框带箭头）
2. 向下滑动，找到 **"添加到主屏幕"**
3. 可以自定义名称，如"轻松记账"
4. 点击右上角 **"添加"**

**Android (Chrome)：**
1. 点击右上角 **三个点菜单**
2. 点击 **"添加到主屏幕"**
3. 自定义名称
4. 点击 **"添加"**

### 3. 开始使用
现在桌面上会出现一个 APP 图标，点击就能直接使用！

## 🔧 常见问题

### Q: 部署需要花钱吗？
A: 完全免费！Vercel、Netlify、GitHub Pages 都提供免费套餐，个人使用足够了。

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地存储（LocalStorage），只要不清除浏览器数据，就会一直保存。建议定期使用导出功能备份。

### Q: 可以在多个设备同步数据吗？
A: 当前版本是纯前端应用，数据保存在本地。如需多设备同步，可以考虑：
- 手动导出数据再导入
- 或者后续升级到带后端数据库的版本

### Q: 语音功能在某些手机不能用？
A: 语音识别需要浏览器支持：
- ✅ iPhone Safari：支持良好
- ✅ Android Chrome：大部分支持
- ❌ 某些老旧浏览器可能不支持，可以用手动输入
