## 🎯 最简单的部署方式（3 步搞定）

### 推荐：使用 Netlify Drop（无需注册，无需代码）

**步骤 1：打开网站**
访问：**https://app.netlify.com/drop**

**步骤 2：拖拽文件夹**
把整个 `expense_tracker` 文件夹拖到网页上的虚线框内

**步骤 3：获取网址**
上传完成后，你会立即获得一个网址，格式类似：
```
https://random-name-12345.netlify.app
```

点击 **"Site settings"** → **"Change site name"** 可以自定义名称，比如：
```
https://weimo-expense-tracker.netlify.app
```

**完成！** 🎉

## 📱 在手机上使用

### iPhone 用户：
1. Safari 浏览器打开你的网址
2. 点击底部 **分享按钮**（方框 + 箭头）
3. 选择 **"添加到主屏幕"**
4. 输入名称"轻松记账"
5. 点击右上角 **"添加"**

### Android 用户：
1. Chrome 浏览器打开你的网址
2. 点击右上角 **三个点菜单**
3. 选择 **"添加到主屏幕"**
4. 输入名称"轻松记账"
5. 点击 **"添加"**

现在桌面上会出现一个 APP 图标，点击就能使用！

## ✨ 核心功能

- 🎤 **语音记账** - 说话就能记录："早餐 15 元"、"打车 25 块"
- ✍️ **手动输入** - 传统输入方式
- 📊 **实时汇总** - 今日/本周/本月自动统计
- 🏷️ **分类管理** - 8 种常见分类（餐饮、交通、购物等）
- 💾 **数据持久化** - 保存在浏览器本地，不会丢失
- 📤 **导出备份** - JSON 格式导出数据
