# 📱 记账 APP 手机使用指南

## 方式一：本地文件直接使用

### 安卓手机
1. 将 `index.html` 文件传到手机（微信/QQ/钉钉均可）
2. 在手机文件管理器中找到该文件
3. 长按 → 打开方式 → 选择浏览器（Chrome/华为浏览器等）
4. 即可开始使用

### iPhone
1. 将 `index.html` 文件传到手机
2. 在微信中点击文件 → 右上角"..." → 存储到"文件"
3. 打开"文件"APP → 找到 `index.html` → 点击用 Safari 打开

---

## 方式二：部署到网上（推荐⭐）

### 使用 Vercel 免费部署（5 分钟搞定）

1. **访问 Vercel**
   - 手机或电脑浏览器打开：https://vercel.com

2. **注册/登录**
   - 用 GitHub 账号一键登录（没有就免费注册一个）

3. **上传项目**
   - 点击 "Add New Project"
   - 把 `expense_tracker` 文件夹拖拽上传
   - 或者连接 GitHub 仓库

4. **获取访问链接**
   - 部署完成后会生成一个 URL，类似：`https://your-app.vercel.app`
   - 复制这个链接到手机浏览器

5. **添加到手机桌面**
   - **安卓 Chrome**：菜单 → "添加到主屏幕"
   - **iPhone Safari**：分享按钮 → "添加到主屏幕"
   - 这样就像原生 APP 一样，点击图标就能打开

## 💡 使用技巧

### 语音记账
- 点击 🎤 麦克风图标
- 直接说："早餐 15 元"、"打车 32 块"、"买衣服 299"
- AI 会自动识别金额和分类

### 查看统计
- 切换顶部标签查看：今日/本周/本月
- 总金额实时汇总
- 分类占比一目了然

### 数据保存
- 所有数据自动保存在手机浏览器本地存储
- 关闭页面再打开，数据还在
- ⚠️ 注意：清除浏览器缓存会删除数据，建议定期导出备份

## 📊 数据导出（备份）

1. 点击页面底部"导出数据"按钮
2. 会下载 JSON 格式的数据文件
3. 建议每月导出一次，存到云盘或电脑

## ❓ 常见问题

**Q: 语音识别不准确怎么办？**
A: 在安静环境下使用，说话清晰一些。如果还是不准，直接用键盘输入。

**Q: 换了手机数据会丢失吗？**
A: 是的，数据保存在原手机浏览器中。换手机前请先导出数据，然后在新手机导入。

**Q: 可以多人一起记账吗？**
A: 当前版本是单机版，不支持多用户。如需共享，可以部署到服务器后分享链接给家人。

**Q: 为什么统计数据和实际不符？**
A: 检查是否清除了浏览器缓存，或者在不同浏览器/设备间切换过。
